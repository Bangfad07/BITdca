# BITdca
kebingungan dan ketidakpastian dalam investasi Bitcoin, kurang gambaran historis DCA, pengaruh emosi, dan jarak antara paham dan mulai investasi. Sehingga solusi yang ditawarkan oleh BitDCA adalah Kalkulator simulasi DCA berbasis data historis dengan parameter lengkap, grafik pertumbuhan portofolio dan edukasi singkat tentang DCA
